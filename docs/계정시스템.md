## ACC-AUTH-001 인증 흐름

```
앱 시작
  ├─ 로그인 계정 0개 → firstSetup (관리자 계정 생성)
  ├─ 유효 세션 존재 → app (자동 로그인)
  └─ 그 외 → login
```

| appState | 조건 | 화면 |
|---|---|---|
| `loading` | 초기 로드 중 | 로딩 스피너 |
| `firstSetup` | `team`에 `password` 가진 멤버 0명 | 관리자 계정 생성 폼 |
| `login` | 로그인 계정 있음, 세션 없음/유효 X | 로그인 화면 |
| `app` | 세션 유효 | 메인 앱 |

## ACC-AUTH-002 로그인 자격증명

- **식별자**: 이메일 또는 이름 (둘 중 하나 일치하면 OK)
- **비밀번호**: 평문 일치 검증 (현재). 운영 시 해싱 필수 → ACC-AUTH-004
- **세션 키**: `session = {userId}` (window.storage)

## ACC-AUTH-003 로그아웃

- 사이드바 하단 "로그아웃" 버튼
- 동작: `session` 키 null, `currentUser` 초기화, `appState`를 `login`으로

## ACC-ROLE-001 역할 체계

### 직급 (grade)

| 직급 | 권한 그룹 |
|---|---|
| `Junior RM` | RM |
| `Mid RM` | RM |
| `Senior RM` | RM |
| `팀장` | 팀장 |
| `관리자` | 관리자 |

### 권한 그룹 매핑 (코드 기준)

```javascript
roleGroup =
  isAdmin || grade === "관리자"  → "관리자"
  grade === "팀장" || grade === "Lead RM"  → "팀장"
  그 외 → "RM"
```

### 관리자 플래그 (`isAdmin`)

- 직급과 별개로 부여 가능
- `관리자` 직급은 자동 `isAdmin=true`
- 다른 직급도 `isAdmin=true`면 관리자 권한 보유

## ACC-ROLE-002 멤버 종류

| 종류 | 필드 | 로그인 | 견적서 RM 지정 |
|---|---|---|---|
| 정식 계정 | `tagOnly: false`, `password: 있음` | ✓ | ✓ |
| 태그 전용 | `tagOnly: true`, `password: 없음` | ✗ | ✓ |

태그 전용은 외부 RM이나 협력자를 견적서에 이름으로만 등록할 때 사용.

## ACC-PERM-001 권한 매트릭스 (DEFAULT_PERMS)

| 메뉴 | RM | 팀장 | 관리자 |
|---|---|---|---|
| 고객관리 | ✓ | ✓ | ✓ |
| 상담관리 | ✓ | ✓ | ✓ |
| 계약관리 | ✗ | ✓ | ✓ |
| 작업자관리 | ✓ | ✓ | ✓ |

**항상 접근 (권한 무관)**:
- 대시보드
- 작업자관리
- 설정 (단, 내부에서 관리자/팀장만 보이는 영역 있음)

**관리자 전용 (설정 내부)**:
- 계정, RM 팀, 직무, 권한, 데이터 탭

**관리자 + 팀장 (설정 내부)**:
- AI 학습 패턴 탭 (REQ-AI-005, REQ-SETTINGS-001) — RM에게는 미노출

비관리자(+ 비팀장)는 설정 진입 시 "내 프로필"만 표시.

## ACC-PERM-002 권한 변경 정책

- 관리자만 권한 매트릭스 수정 가능
- 변경은 즉시 반영 (사이드바 메뉴 표시 갱신)
- 권한 박탈 시 해당 메뉴가 사이드바에서 제거됨

## ACC-DATA-001 데이터 분리 키

```
customers:{userId}    → 사용자별 고객 데이터
notes:{customerId}    → 고객별 상담 노트
```

## ACC-DATA-002 공유 데이터

```
teams        → 팀 멤버 목록 (전역)
rates        → 시급 테이블 (전역)
categories   → 직무 카테고리 (전역)
perms        → 권한 매트릭스 (전역)
session      → 현재 세션 (브라우저별)
```

**제약**: 현재는 단일 기기 단일 세션 운영. 멀티 사용자 동시 접속 미지원.
